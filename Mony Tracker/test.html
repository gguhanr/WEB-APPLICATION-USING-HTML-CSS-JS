Google Sheets Integration

Sync your wallet data to your own Google Sheet. Follow the 5 steps below — takes about 2 minutes!

1

Create a new Google Sheet

Open sheets.new
2

Open Apps Script

In your sheet → click Extensions → Apps Script

3

Paste the Connector Script

Delete all existing code, paste this script, then click Save (💾)

4

Deploy as Web App

Click Deploy → New Deployment, then set:

Type: Web App
Execute as: Me
Who has access: Anyone

Click Deploy → allow permissions → Copy the Web App URL

5

Paste your Web App URL here
